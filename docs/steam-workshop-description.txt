Let an AI play Timberborn. Full read/write HTTP API with a Python client so Claude, ChatGPT, or your own scripts can run a beaver colony.

[h1]How it works[/h1]
The mod runs an HTTP server (port 8085) inside the game. Python client included:

[code]
python timberbot.py summary
python timberbot.py buildings
python timberbot.py place_building prefab:LumberjackFlag.IronTeeth x:120 y:130 z:2 orientation:south
python timberbot.py plant_crop x1:123 y1:138 x2:128 y2:141 z:2 crop:Kohlrabi
python timberbot.py beavers                          # beaver wellbeing + critical needs
python timberbot.py scan x:120 y:140 radius:10        # compact map for AI
python timberbot.py visual x:120 y:140 radius:10  # colored ASCII map for humans
python timberbot.py watch
[/code]

[h1]Getting started[/h1]
1. Subscribe and enable in Mod Manager
2. Start a game. API starts automatically on port 8085
3. Find timberbot.py in your mods folder (Documents\Timberborn\Mods\Timberbot\)
4. Run: python timberbot.py ping
5. Run: python timberbot.py (lists all commands)
