Tell Claude "keep my beavers alive and get to 77 wellbeing" and watch it figure out the rest. It reads your colony, places buildings, routes paths up cliffs, manages workers, and reacts to droughts. You play together.

[code]
Open the Timberbot API widget, click Settings, pick Claude or Codex, click Start
[/code]

You need Python and a couple of pip packages (requests, toons) installed first. After that, everything runs from inside the game. The widget opens a terminal session where you and the AI talk through what to do next.

[h1]What can the AI actually do[/h1]
It sees everything you see. Every beaver's position, needs, and workplace. Every building's workers, power, inventory, and recipes. The full map with terrain, water, trees, and what's occupying each tile.

It can place and demolish buildings, route paths with auto-stairs across elevation changes, assign workers, manage crops and trees, control floodgates, adjust stockpiles, unlock science, and change game speed. All through the same validation the game uses when you place things by hand.

[code]
timberbot.py brain
timberbot.py top
timberbot.py place_path x1:0 y1:0 x2:200 y2:180
timberbot.py map x1:110 y1:130 x2:130 y2:150
[/code]

Ask it to build a lumber district and it will find valid spots near trees, place the buildings, connect paths back to the district center, and assign workers. Ask it to prepare for drought and it will check water reserves, adjust production, and build storage.

[h1]Not just for AI[/h1]
The whole thing is an HTTP API. You can write your own scripts, build dashboards, or hook it into whatever you want. 68 webhook events let your code react to droughts, deaths, construction, and more without polling. There's a Python client included, but anything that speaks HTTP works.

[h1]Getting started[/h1]
1. Subscribe and enable in Mod Manager
2. Start a game
3. Click the Timberbot widget in the corner, pick your AI, hit Start

Works with Claude Code, OpenAI Codex, or any CLI agent that can make HTTP calls. See the [url=https://abix-.github.io/TimberbornMods/getting-started/]full setup guide[/url] for details.
