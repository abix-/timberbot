Play Timberborn with an AI. Timberbot API gives Claude, ChatGPT, or your own scripts full read/write access to your colony over HTTP so they can partner with you to run it.

[h1]What Timberbot API adds[/h1]
Timberborn already has a built-in HTTP API (port 8080) for automation with levers and adapters. Timberbot API (port 8085) adds everything the built-in API doesn't cover:

[table]
[tr][th]Capability[/th][th]Built-in API[/th][th]Timberbot API[/th][/tr]
[tr][td]Building placement + demolition[/td][td]No[/td][td]Yes, with tile validation[/td][/tr]
[tr][td]Read game state[/td][td]On/off signals via sensor -> adapter chains[/td][td]Full data: buildings, beavers, trees, map, resources, weather, science[/td][/tr]
[tr][td]Beaver wellbeing + needs[/td][td]No[/td][td]Per-beaver position, district, needs, carried goods, bot durability. detail:full for all 38 needs with group category[/td][/tr]
[tr][td]Colony management[/td][td]No[/td][td]Workers, priority, speed, crops, trees, science, distribution[/td][/tr]
[tr][td]Map + terrain reading[/td][td]No[/td][td]Terrain, water, contamination, occupants[/td][/tr]
[tr][td]AI-optimized output[/td][td]No[/td][td]TOON format, scan, visual map with terrain height, alerts, tree clusters[/td][/tr]
[/table]

Use both together: the built-in API handles real-time automation (sensor -> lever -> floodgate). Timberbot handles strategic AI decisions (check needs -> place farmhouse -> plant crops).

[h1]How it works[/h1]
Timberbot API runs an HTTP server (port 8085) inside the game. Python client included:
[code]
python timberbot.py summary                          # colony snapshot: resources, weather, alerts, projections
python timberbot.py beavers                          # position, district, wellbeing, carried goods per beaver
python timberbot.py beavers detail:full              # all 38 needs with group category per beaver
python timberbot.py buildings detail:full             # inventory, production progress, power, effect radius
python timberbot.py power                            # power networks with supply vs demand
python timberbot.py visual x:120 y:140 radius:10     # ASCII map with terrain height shading
python timberbot.py place_building prefab:FarmHouse.IronTeeth x:120 y:130 z:2 orientation:south
[/code]

[h1]Getting started[/h1]
1. Subscribe and enable in Mod Manager
2. Start a game. API starts automatically on port 8085
3. Install dependencies: pip install requests toons
4. Find timberbot.py in your mods folder (Documents\Timberborn\Mods\Timberbot\)
5. Run: python timberbot.py ping
6. Run: python timberbot.py (lists all commands)

[h1]Let AI play for you[/h1]
The mod includes an AI prompt (docs/timberbot.md) that teaches Claude, ChatGPT, or any LLM how to run your colony. With Claude Code:

[code]
# copy the prompt as a skill
cp docs/timberbot.md ~/.claude/skills/timberbot/SKILL.md

# let Claude play on a loop
/loop 1m /timberbot
[/code]

The AI will check beaver wellbeing, manage food and water, place buildings, mark trees, and keep your colony alive.
