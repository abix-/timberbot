Let an AI play Timberborn. Full read/write HTTP API with a Python client so Claude, ChatGPT, or your own scripts can run a beaver colony.

[h1]Timberbot vs Vanilla HTTP API[/h1]
Timberborn has a built-in HTTP API (port 8080) for automation with levers and adapters. Timberbot (port 8085) adds everything else. They work together:

[table]
[tr][th]What[/th][th]Vanilla HTTP (port 8080)[/th][th]Timberbot (port 8085)[/th][/tr]
[tr][td]Floodgates[/td][td]Control via levers[/td][td]Set height directly[/td][/tr]
[tr][td]Sensors (depth, flow, power)[/td][td]Read via adapters[/td][td]Read all game state directly, no sensors needed[/td][/tr]
[tr][td]Real-time automation[/td][td]Sensor -> logic -> lever chains[/td][td]Not real-time, polled by AI[/td][/tr]
[tr][td]Building placement[/td][td]No[/td][td]Yes, with validation[/td][/tr]
[tr][td]Beaver wellbeing[/td][td]No[/td][td]Yes, per-beaver needs[/td][/tr]
[tr][td]Colony management[/td][td]No[/td][td]Workers, priority, speed, crops, trees, science[/td][/tr]
[tr][td]Map reading[/td][td]No[/td][td]Terrain, water, occupants, scan[/td][/tr]
[/table]

Use vanilla HTTP for real-time automation triggers (water rises -> open floodgate). Use Timberbot for strategic AI decisions (need food -> place farmhouse). The Python client wraps both APIs in one place.

[h1]How it works[/h1]
Two APIs, one Python client:
[code]
# Timberbot API (port 8085) -- strategic control
python timberbot.py summary
python timberbot.py buildings
python timberbot.py beavers
python timberbot.py place_building prefab:FarmHouse.IronTeeth x:120 y:130 z:2 orientation:south
python timberbot.py scan x:120 y:140 radius:10

# Vanilla API (port 8080) -- real-time automation
python timberbot.py levers
python timberbot.py lever_on name:MyFloodgateLever
[/code]

[h1]Getting started[/h1]
1. Subscribe and enable in Mod Manager
2. Start a game. API starts automatically on port 8085
3. Find timberbot.py in your mods folder (Documents\Timberborn\Mods\Timberbot\)
4. Run: python timberbot.py ping
5. Run: python timberbot.py (lists all commands)

[h1]Let AI play for you[/h1]
The mod includes an AI playbook (docs/timberbot.md) that teaches Claude, ChatGPT, or any LLM how to run your colony. With Claude Code:

[code]
# copy the playbook as a skill
cp docs/timberbot.md ~/.claude/skills/timberbot/SKILL.md

# let Claude play on a loop
/loop 1m /timberbot
[/code]

The AI will check beaver wellbeing, manage food and water, place buildings, mark trees, and keep your colony alive.
