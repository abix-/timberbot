Play Timberborn with an AI. Full read/write HTTP API with a Python client so Claude, ChatGPT, or your own scripts can partner with you to run a beaver colony.

[h1]What Timberbot adds[/h1]
Timberborn already has a built-in HTTP API (port 8080) for automation with levers and adapters. Timberbot (port 8085) adds everything the built-in API doesn't cover:

[table]
[tr][th]Capability[/th][th]Built-in API[/th][th]Timberbot API[/th][/tr]
[tr][td]Building placement + demolition[/td][td]No[/td][td]Yes, with tile validation[/td][/tr]
[tr][td]Read all game state[/td][td]Lever/adapter signals only[/td][td]Buildings, beavers, trees, map, resources, weather[/td][/tr]
[tr][td]Beaver wellbeing + needs[/td][td]No[/td][td]Per-beaver needs, contamination, workplace[/td][/tr]
[tr][td]Colony management[/td][td]No[/td][td]Workers, priority, speed, crops, trees, science, distribution[/td][/tr]
[tr][td]Map + terrain reading[/td][td]No[/td][td]Terrain, water, contamination, occupants[/td][/tr]
[tr][td]AI-optimized output[/td][td]No[/td][td]TOON format, scan, visual, alerts, tree clusters[/td][/tr]
[/table]

Use both together: the built-in API handles real-time automation (sensor -> lever -> floodgate). Timberbot handles strategic AI decisions (check needs -> place farmhouse -> plant crops).

[h1]How it works[/h1]
The mod runs an HTTP server (port 8085) inside the game. Python client included:
[code]
python timberbot.py summary
python timberbot.py buildings
python timberbot.py beavers
python timberbot.py place_building prefab:FarmHouse.IronTeeth x:120 y:130 z:2 orientation:south
python timberbot.py scan x:120 y:140 radius:10
python timberbot.py alerts
[/code]

[h1]Getting started[/h1]
1. Subscribe and enable in Mod Manager
2. Start a game. API starts automatically on port 8085
3. Find timberbot.py in your mods folder (Documents\Timberborn\Mods\Timberbot\)
4. Run: python timberbot.py ping
5. Run: python timberbot.py (lists all commands)

[h1]Let AI play for you[/h1]
The mod includes an AI playbook (docs/timberbot.md) that teaches Claude, ChatGPT, or any LLM how to run your colony. With Claude Code:

[code]
# copy the playbook as a skill
cp docs/timberbot.md ~/.claude/skills/timberbot/SKILL.md

# let Claude play on a loop
/loop 1m /timberbot
[/code]

The AI will check beaver wellbeing, manage food and water, place buildings, mark trees, and keep your colony alive.
